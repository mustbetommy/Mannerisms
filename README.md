# Mannerisms

**Your character has a personality now.**

Mannerisms is a Dalamud plugin that brings your FFXIV character to life. Set up gestures that suggest emotes based on what you type in chat: type "bye" and get a wave suggestion, say "WHAT" and see the aback emote ready to fire. One click or keybind to accept.

## Features

### Gesture Suggestions

Mannerisms watches what you type in chat and suggests matching emotes in a floating picker window. You stay in control: suggestions appear, you decide whether to use them.

- **Three trigger modes**: Common (pre-made toggleable patterns), Simple (build patterns with a comma-separated word list), and Advanced (full regex)
- **Per-character profiles** with independent gesture sets
- **Target-aware triggers** that only fire when you have a target, or when your target matches the recipient of a tell
- **Party-aware triggers** that verify your target is in your party
- **Configurable keybinds** to accept or dismiss suggestions without touching the mouse
- **Suggestion timeout** with visual countdown: suggestions fade if you don't act on them

## Commands

| Command | Description |
|---|---|
| `/manners` | Open the configuration window |

## Installation

Mannerisms is available in the Dalamud Plugin Installer. Open `/xlplugins` and search for **Mannerisms**.
